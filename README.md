# ✈️ المساعد الذكي لمكتب دروب الياسين للسفريات | AI Travel Agent

## 📌 عن المشروع (About The Project)
نظام خدمة عملاء ذكي (AI Customer Service Agent) مبني خصيصاً لمكتب "دروب الياسين للسفريات والسياحة". يهدف المشروع إلى أتمتة الردود على استفسارات العملاء بدقة عالية باستخدام أحدث تقنيات الذكاء الاصطناعي التوليدي، مع تجنب مشكلة الهلوسة (Hallucination) المنتشرة في النماذج اللغوية.

## 🚀 التقنيات المستخدمة (Technical Architecture)
تم بناء هذا المشروع وتصميمه باستخدام تقنيات متقدمة في مجال الذكاء الاصطناعي:
* **LLM Engine:** الاعتماد على نموذج **Google Gemini 2.5 Flash** لسرعة الاستجابة ودقة الفهم للهجة المحلية.
* **RAG (Retrieval-Augmented Generation):** ربط النموذج بقاعدة بيانات محلية (Knowledge Base) لضمان استخراج أسعار دقيقة وموثوقة لخدمات الطيران والجوازات.
* **Hybrid Routing System:** نظام توجيه ذكي يستخدم التعابير المنطقية (Regex) للرد اللحظي على التحيات، وتقليل استدعاء الـ API لتقليل التكلفة وزيادة السرعة.
* **Cloud Deployment:** تم نشر النظام سحابياً عبر منصة **Streamlit Cloud** لتجاوز قيود الحظر الجغرافي (VPN) وتوفير وصول سهل للمستخدمين.

## 🛠️ لغات وأدوات التطوير (Tech Stack)
* **اللغة الأساسية:** Python 3
* **واجهة المستخدم:** Streamlit
* **الذكاء الاصطناعي:** Google Generative AI API

## 📋 المميزات الأساسية (Key Features)
1. ردود فورية ودقيقة تعتمد على ملف بيانات الأسعار `travel_data.txt`.
2. واجهة مستخدم (UI) احترافية ومتجاوبة.
3. إمكانية تحميل تفاصيل المحادثة كإيصال أو مسودة للعميل.
4. القدرة على فهم اللهجة اليمنية البيضاء بسلاسة.
